# Chatbot Frontend

A simple Next.js chatbot interface with TypeScript and Tailwind CSS that displays streaming responses.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Features

- Clean, responsive chat interface
- Real-time streaming responses from the backend
- Built with Next.js 15, React 19, and TypeScript
- Styled with Tailwind CSS

## Configuration

The frontend connects to the backend at `http://localhost:5000`. Make sure the backend server is running before using the chatbot.
